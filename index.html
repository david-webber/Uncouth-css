Uncouth CSS

A(nother) minimal, classless css library.

Yeah, but why?

I found myself applying the same base styles to a lot of demos and new projects I wa creating. Use this to improve markup sites, or use as it as a base style for your next project! View on GitHub

Themes

There are currently two themes 'light' and 'dark', the main-auto.css file will automatically detect the preferred browser theme.
